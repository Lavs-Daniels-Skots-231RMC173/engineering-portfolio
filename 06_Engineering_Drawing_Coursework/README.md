## EN — for GitHub README / EN CV

**Engineering Drawing Coursework (RTU, 2024)**

Four technical drawings produced as part of foundational engineering-drawing coursework. Each follows ISO/LVS conventions: proper view selection, section views, dimensioning per standard, title-block compliance and material specification.

**Drawings**
- **Balsts (Support)** — A4, scale 1:1 — bracket / mounting support component
- **Statne (Frame)** — A3, scale 1:1 — frame/stand component
- **Detail (A3)** — A3, scale 1:1 — third detail drawing
- **Detail (A4, Task 5)** — A4, scale 1:1 — fifth-task detail drawing

**Skills demonstrated:** technical drawing (ISO/LVS conventions) · multi-view projection · section views · dimensioning and tolerancing · title-block discipline · A3/A4 sheet layout

These drawings form the foundation that the later RTK 4.7 part drawing and metrology study work build on — same standards, more complex parts.
