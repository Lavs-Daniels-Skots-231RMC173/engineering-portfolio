# DE0807_24-25-P-practical-task-02-questions

Sometimes the main file that is launched is not called main.py, that is completely normal.

While the file might not be called main, knowing that multiple files can be interconnected, the interpretter assigns a status main to the file in which you press the play/launch button. This status can be checked in program to decide what to do. This method is often used to check if you are running tests or "the live version". 

## Task

1. Read all the comments, they have information about syntax and structure of the file
2. Cast data types to correct ones
3. Compare values and return the correct message

This task has 2 types of autograding - input/output and function tests.
