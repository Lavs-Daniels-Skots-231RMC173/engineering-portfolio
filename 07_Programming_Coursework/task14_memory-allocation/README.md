# DIP750_24P_ENG_T18

Imagine that you need to read a lot of data in memory, a dynamic array is more efficient with large amounts of data.

Your task is to read plates.txt file using a buffer and char array.

After task check with *valgrind* for memory leaks.

Suggested solution is malloc, but feel free to try out other memory assignment methods.


Additional material:

malloc - https://en.cppreference.com/w/c/memory/malloc

calloc - https://en.cppreference.com/w/c/memory/calloc

realloc - https://en.cppreference.com/w/c/memory/realloc

free - https://en.cppreference.com/w/c/memory/free

alloc is part an old library, which is rarely used anymore, so it's not suggested for this task.
