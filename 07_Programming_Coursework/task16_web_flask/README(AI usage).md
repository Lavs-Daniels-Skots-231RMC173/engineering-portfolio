**1. Notes About AI Use**
While building the Spotify-themed site, I ran into a few places where I wasn’t sure how to structure my HTML templates (index, input, output) or how to fix small routing errors in Flask. I described those problems briefly to ChatGPT, and it suggested the correct route decorators and helped me adjust the Jinja syntax. It also showed me simple CSS tweaks—like creating a Spotify-style “pill” button and adding hover effects. In short, AI guided me through fixing template mistakes and styling choices without having to dig through official docs line by line.

**2. Thoughts About AI in Development**
AI can be a very helpful “pair programmer.” When I hit a 405 method error or a template syntax issue—I could explain the problem in plain English and quickly see examples of how to fix it. It sped up my workflow because I didn’t need to search through forum posts. That said, I always double-checked the suggestions and made sure I understood why each change worked.

**3. Thoughts About AI in Learning**
Using AI helped me learn new concepts faster. Instead of copy-pasting entire solutions, I got short explanations and examples that made sense in the context of my own code. This meant I actively thought about each change, which helped reinforce what I was learning. Overall, AI felt like an on-demand tutor that would clarify confusing topics as I worked.

**4. Tackling a Project in an Unknown Technology**
It’s definitely possible to start a project in something unfamiliar (for example, mixing Flask with Jinja and SQLite for the first time). AI made it easier because I could describe the error or ask how to set up a new route, and then get a concise answer. Having that guidance gave me confidence to keep building even when I wasn’t 100% sure how things should work under the hood.

**5. Three Skills I Learned**

1. **Flask Routing & Jinja Templating**
   I practiced setting up multiple routes, passing form data via query parameters, and using Jinja logic to show different content.
2. **Custom CSS with Bootstrap**
   I learned how to take a Bootstrap “button” or card and customize it (colors, hover animations, pill-shaped buttons) to match a cohesive Spotify-inspired design.
**3. Dynamic Embedding of External Media in Flask**
    I learned how to embed a third-party music player into a Flask/Jinja template by extracting a track ID from a URL and constructing an `<iframe>` dynamically. This taught me how to integrate external content into my site’s design and ensure it displays correctly within Bootstrap cards.
