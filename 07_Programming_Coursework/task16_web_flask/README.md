# web project

For this to work, in terminal write pip install flask


To launch code, in terminal write run flask wich will offer to open a page in new tab - this page is your result.

To stop flask - click in terminal and press ctrl+c .


You're task is to create a web page with:
* About page 
  * Text
  * Picture
  * Link
* Input form
  * At least one input field
  * At least one dropdown field
  * At least one check box
  * Submit button
* Page that displays saved data
* Additional page about whatever you want, that displays something not listed in this task


Mandatory elements:
* Bootstrap
* JS
* Table formatting 
* Working web site w/o errors (warnings are allowed)
* at leats 3 elements needs to have custom CSS (you are allowed to customize Bootstrap element)


Languages to use:
* Python - to run and control web app
  * each address has a function that controls it
* Jinja - templating language
  * includes information from python into HTML template
* HTML - skeleton for web page
  * contains all the elements
* CSS - styling of web page
* JavaScript - interactivity


form_example.html is not supposed to be edited, it's a sample of how to do a form.

Additional information:
* you need to include the creation of db command when opening the app for first time as web pgae (so every time run flask), this is an technical issue with access to data when launching outside of your original codespace
* check that it works outside of codespace
