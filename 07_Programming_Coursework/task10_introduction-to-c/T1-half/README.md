## To run the file

First of all you need the extension added. This language compiles / builds files - meaning that your *.c/*.cpp program will be converted to launchable file (similar to *.exe but with no extension).

When you get the play button, besides it there is a triangle - choose build / run command not debug. Debug mode is meant when you need to find a value mistake and create break points and monitored values. When you see in file list a file with same name, but no extension - that is your result. Now you can either click play button again (only on the first time it needs to be done twice) or open terminam and write ./mainProgram (for this task) and enter. This will run the file.
