# Practical_task_13_search_n_sort

Make following algorithms:
* bubble sort using ( total 1+2p)
    * with linked lists and known element count (1 point) 
    * with linked lists and unknown element count (2 points)
* selection sort with arrays (1 point)
* linear search (total 1+2p)
    * with arrays (1 point)
    * with linked lists ( 2 points)
* binary search with arrays (3 points)

Use of AI tools forbidden unless stated otherwise in algorithm file.
